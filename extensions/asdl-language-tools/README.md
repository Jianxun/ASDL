# ASDL Language Tools

Fresh VS Code/Cursor extension scaffold for `.asdl` authoring.

## Included in this scaffold
- ASDL language registration (`.asdl` extension).
- YAML-compatible TextMate grammar with ASDL overlays for net-first syntax.
- Baseline snippets for file/module/device authoring.

## Development
```bash
npm install
npm run compile
```
