# Refactor Specs

Draft specifications for the in-progress architecture refactor. These docs
describe the proposed dataclass core, pattern service, and parallel stack while
the legacy xDSL pipeline remains functional.

Status: Informational only; not canonical.

## Files
- `spec_refactor_patterned_graph.md`: PatternedGraph core model + registries.
- `spec_refactor_pattern_service.md`: pattern parsing/binding/expansion rules.
- `spec_refactor_stack.md`: parallel pipeline and API/CLI boundaries.
- `spec_refactor_migration.md`: phased migration plan and parity strategy.
